Olá! Seja bem-vindo(a) ao meu app! 😊 Antes de avançar, tem algumas informações muito importantes que você precisa saber:

⚠️ Instalação de dependências:
Se você deu um git clone ou fez um fork, certifique-se de que todos os arquivos foram baixados corretamente.

Caso tenha dúvidas, rode o comando:
npm i

Isso vai instalar todos os pacotes necessários.
Como eu usei muitos pacotes para testes e funcionalidades avançadas, pode ser que você precise rodar esse comando mais de uma vez.
